## Deskripsi

Program ini merupakan implementasi pembuatan objek **mobil** menggunakan **OpenGL** dengan bantuan library **GLUT**.

Objek mobil digambar menggunakan kombinasi beberapa **primitive drawing** OpenGL seperti:

* Garis
* Segitiga
* Lingkaran

Adapun **stack** digunakan untuk menggabungkan seluruh primitive drawing.
